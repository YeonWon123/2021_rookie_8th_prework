# ex1 : 처음 생성해 보는 Spring Project

* 프로젝트 설정 : Gradle 프로젝트  
* Group : 'org.zerock'  
* Artifact : 'ex1'  
* Packaging : 'war'  
* JDK ver : 11  
* 프로젝트에 추가하는 의존성 항목 : Spring Boot DevTools, Lombok, Spring Web  
* SampleController 클래스 추가  
* @RestController를 이용해서 별도의 화면 없이 데이터를 전송  
* @GetMapping을 이용해서 브라우저의 주소창에서 호출이 가능하도록 설정
